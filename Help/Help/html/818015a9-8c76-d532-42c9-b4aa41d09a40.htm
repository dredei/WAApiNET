WAApiException - свойства

Тип WAApiException предоставляет следующие члены.

Свойства
  ИмяОписание
Data
Возвращает коллекцию пар "ключ-значение", предоставляющих дополнительную заданную пользователем информацию об исключении.
(Унаследован от Exception.)
Exception
Исключение
HelpLink
Возвращает или задает ссылку на файл справки, связанный с этим исключением.
(Унаследован от Exception.)
HResult
Возвращает или задает HRESULT — кодированное числовое значение, присвоенное определенному исключению.
(Унаследован от Exception.)
InnerException
Возвращает экземпляр класса Exception, вызвавший текущее исключение.
(Унаследован от Exception.)
JSONAnswer
JSON ответ
JSONQuery
JSON запрос
Message
Возвращает сообщение, которое описывает текущее исключение.
(Унаследован от Exception.)
Source
Возвращает или задает имя приложения или объекта, вызывавшего ошибку.
(Унаследован от Exception.)
StackTrace
Получает строковое представление непосредственных кадров в стеке вызова.
(Унаследован от Exception.)
TargetSite
Возвращает метод, выбрасывающий текущее исключение.
(Унаследован от Exception.)
